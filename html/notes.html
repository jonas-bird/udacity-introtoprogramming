Lesson 1: Styling with CSS

When the browser renders the DOM the block level elements are rendered as boxes.

Each box has:

  1. Padding between the content of the element and the border
  2. A border (by default not visible, but it can be made wider and displayed)
  3. A margin that controls how close other elements are displayed. Negative values can be used to create overlap
